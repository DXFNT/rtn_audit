Retention & emailing audity pre Dexfinity klientov

Live audity účtov klientov — výkonnosť, RFM segmenty, retenčné kohorty, databáza, automatizácie a kalendár kampaní. Dáta priamo z používaných nástrojov (Leadhub, Ecomail, ďalšie). Každý audit obsahuje konkrétne odporúčania, harmonogram a porovnanie s odvetvovými štandardmi.

Workflow

Ako pridať nového klienta

Štyri kroky od dát po publikovanie. Detaily v README.md.

1

Vytvor folder

V repe pridaj zložku s názvom klienta lowercase, napr. noviclient/.

2

Pridaj report

Hlavný report ulož ako index.html. Použi DXFNT brand template (Poppins+Questrial, --dex-blue/--dex-navy, Go Beyond).

3

Pridaj entry do landing-u

Otvor index.html v root-e, do const CLIENTS = [...] pridaj objekt s metadátami klienta.

4

Push do main

GitHub Pages publikuje automaticky. URL: dxfnt.github.io/rtn_audit/noviclient/
